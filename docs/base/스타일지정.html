스타일 지정 방법

HTML에 스타일을 적용하기 위해서는 선택자를 지정하여야 합니다.

CSS 구문

선택한것 {
    속성 : 값
}

선택한 것에 대해서 스타일을 적용하게 됩니다. 스타일은 속성과 값을 콜론(:)으로 구분하여 한쌍을 이루게 됩니다. 이렇게 한쌍으로 구성된 것을 스타일 블럭이라고 합니다.

스타일 블럭구분

하나의 스타일은 다수의 스타일 블럭을 적용할 수 있습니다. 여러 블럭을 적용을 할때에는 세미콜론(;)으로 구분을 합니다.

선택한것 {
    속성 : 값;
    속성 : 값;
    속성 : 값;
}

마지막 블럭의 세미콜론은 생략이 가능합니다.
