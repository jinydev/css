선택자

CSS는 html의 요소를 선택하여 스타일을 적용합니다. 따라서 요소를 선택이 필요 합니다.

다양한 선택자들이 존재합니다.

전체 선택자

전체의 요소를 선택할때에는 *를 사용합니다.

<style>
* {
    color:red;
}
</style>

테그 선택

html의 테그를 선택할 수 있습니다. 테그의 이름을 입력하면 됩니다.

h1 {
    color:red;
}

테그연결

복수의 테그를 선택하여 스타일을 적용할 수 있습니다. 여러개의 테그를 동시에 선택할때에는 콤마(,)를 이용하여 구분을 합니다.

h1, h2 {
    color:red;
}
