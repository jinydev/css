위치

요소의 위치를 조정할 수 있습니다. 위애서 아래, 좌 우로 배치를 하게 됩니다.

위치속성

position

  • absolute

첫번째 요소는 body 마진에 영향을 받습니다. 절대 좌표가 적용된 요소들은 다른 요소들의 위치 영향을 주지 않습니다. 화면 전체를 기준으로 위치를 결정합니다.

  • relative 부모 테그의 위치를 기준으로 설정합니다.

left/top/right/bottom

z-index

요소가 겹쳐서 보여질때, 요소의 순서를 변경합니다.

overflow

자신의 위치를 넘을때.

  • none overflow: none;

  • scroll overflow: scroll;
