## 텍스트 스타일

### 줄간격
`line-height` 속성은 여러 텍스트의 줄간격을 지정합니다.

줄간격은 텍스트의 중심선을 기준으로 간격을 처리합니다.

### 들여쓰기
기본값은 0 입니다.

text-indent:10px

### 자간
letter-spacing:0;
기본값은 0 입니다.

### 꾸밈

* none
* line-through
